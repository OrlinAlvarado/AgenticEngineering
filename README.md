# Agentic Engineering

Master the art of building autonomous AI agents and integrating them into software development workflows. This certification demonstrates your expertise in agent architecture, development frameworks, and AI-powered software engineering.

---

## Prerequisites

| Requirement | Details |
|---|---|
| **AI Coding Assistant** | Access to an AI coding assistant (Claude Code, GitHub Copilot, Cursor, or similar) |
| **Programming Fundamentals** | Basic understanding of programming concepts and at least one programming language |
| **Development Environment** | A code editor and terminal setup for hands-on exercises |
